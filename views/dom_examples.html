DOM Examples

To begin playing with the DOM in JavaScript/jQuery we are going to play with a simple example using boxes.

  • Start by creating a boxes_example project folder on your desktop. Open the project in sublime.

  • Create a file called boxes.html in the project folder.

  • Create a folder called public in the project folder.

    • In the public folder create a stylesheets folder.

      • In the stylesheets folder create a boxes.css file

    • In the public folder create a javascripts folder.

      • In the javascripts folder create a jquery.js file.
      • In the javascripts folder create a boxes.js file.
